This easy-to-use plugin allows WooCommerce store owners to create WooCommerce tags in bulk. 

== Description ==

This plugin allows WooCommerce store owner to create WooCommerce tags in bulk. You just have enter tags separated by commas and parent tag in which you want to add the all those tags. 

== Installation ==

Method 1:
<ol>
<li> Download the plugin bulk-wc-tag-creator.zip </li>
<li> Unpack the bulk-wc-tag-creator.zip file and extract the bulk-wc-tag-creator folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the Bulk WooCommerce Tag Creator and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin bulk-wc-tag-creator.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded bulk-wc-tag-creator.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
